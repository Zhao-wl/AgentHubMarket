# Test Hello

> 当前版本：**v0.2.0** — 用于验证插件市场的版本更新链路。

AgentHub 官方市场仓库内置的最小测试插件，用于验证插件市场的完整链路：

- 市场列表展示与详情页 README 渲染
- 从 GitHub 归档下载并安装
- 项目级启用 / 禁用
- 活动栏图标与左侧占位视图
- 设置中心配置项贡献

## 启用后如何验证

1. 在插件市场中安装 **Test Hello**。
2. 在当前项目详情中启用该插件。
3. 左侧活动栏应出现 **Hello Test** 图标。
4. 点击图标，左侧 L 面板应显示占位视图，包含 `pluginId`、`activityId`、`viewId`。
5. 打开设置中心 → **插件** → **Test Hello**，可看到「问候语」配置项。

## 已知限制

- M1 阶段不加载插件真实 `main` 入口，仅验证 manifest 声明与宿主占位渲染。
- 本插件与市场索引同仓维护；安装时会下载整个 `AgentHubMarket` 仓库归档（含 `marketplace.json` 等索引文件），仅用于开发测试。
